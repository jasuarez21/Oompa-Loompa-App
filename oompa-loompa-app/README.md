# Oompa-Loompa-App

## Description

This is a Oompa Loompas application to render a list in React.
It was built as a project to show the workers in Willy Wonka's factory, to help RRHH department. 

## Installation
- Clone the repo in your terminal by clicking the _green_ clone or download button at the top right and copying the url
- In your terminal, type ```git clone URL```
  - replace URL with the url you copied
  - hit enter
- This will copy all the files from this repo down to your computer
- In your terminal, cd into the directory you just created
- Type ```npm install``` to install all dependencies
- Last, but not least, type ```npm start``` to run the app locally.
- To view the page, go to ```http://localhost:3000/```
- To run the  test, use ```npm test``` in your terminal

- To look at the code, just open up the project in your favorite code editor!
